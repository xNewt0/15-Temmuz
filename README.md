## ✨ Özellikler

| Özellik | Açıklama |
|---|---|
| 🎥 **Video Scroll Efekti** | Scroll pozisyonuna göre `video.currentTime` kontrolü ile kamera hareketi hissi |
| 🕊️ **Şehitler Bölümü** | Alttan yukarı fade + stagger animasyonlu kart grid'i |
| 🎨 **Bayrak Paleti** | Siyah – Kırmızı (`#E30A17`) – Beyaz tonlarında sade, güçlü tasarım |
| 🔤 **Özgün Tipografi** | Başlıklarda `Bebas Neue`, gövde metinde `Inter` |
| 📱 **Tam Responsive** | Mobil, tablet ve masaüstünde sorunsuz deneyim |
| ⚡ **Performans Odaklı** | Optimize edilmiş video/görseller, gecikmesiz scroll |

## 🛠️ Kurulum

```bash
git clone https://github.com/xNewt0/15-Temmuz.git
cd 15-temmuz-site
npm install
npm run dev
```

Build almak için:

```bash
npm run build
```

---

## 📦 Kullanılan Teknolojiler

- **React 18** — UI kütüphanesi
- **Vite** — build tool
- **Tailwind CSS** — stil
- **GSAP + ScrollTrigger** — scroll animasyonları
